# VideoManager-API

Esta é uma API desenvolvida em .NET 10 (C#) para o consumo e postagem de vídeos em html e android.


 ## Funcionalidades 

- [ ] Cadastro e autenticação de conta
- [ ] Edição e exclusão de conta
- [ ] Postagem de vídeos 
- [ ] Listagem de vídeos
- [ ] Edição e exclusão de videos

 ## Tecnologias Utilizadas

- **Linguagem:** C#
- **Framework:** ASP.NET Core Web API (.NET 10)
- **Ferramentas** Visual Studio 2026, Git/GitHub, Microsoft Sql Server Management Studio 22
